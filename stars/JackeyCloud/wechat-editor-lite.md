# ☀️ wechat-editor-lite

> 一款专为自媒体人打造的**极简**、**沉浸式**公众号排版工具。

[在线演示](https://www.1da.top/tools/editor/) | [提交反馈](https://github.com/JackeyCloud/wechat-editor-lite/issues)



## ✨ 核心特性

wechat-editor-lite 旨在解决传统编辑器“功能过载”和“排版繁琐”的痛点，提供“写完即排完”的流畅体验：

- **🎨 极简设计**：无干扰的写作界面，左侧大纲，右侧实时手机预览。
- **⚡️ 斜杠命令 (Slash Command)**：支持 Notion 风格的 `/` 指令，快速插入标题、引用、图片等，无需离开键盘。
- **📝 Markdown 友好**：支持直接粘贴 Markdown 文本，自动渲染为精美排版。
- **📱 公众号完美适配**：
  - 严格遵循 30px 行高与标准字号，确保手机端阅读体验最佳。
  - **一键复制**：保留所有样式（包括图片）直接粘贴到微信公众号后台，格式不乱。
- **💾 本地数据安全**：所有内容实时保存在浏览器本地 (LocalStorage)，无惧意外关闭，隐私绝对安全（无后端上传）。
- **🖼 智能图片处理**：支持 Base64 粘贴与本地上传，复制到公众号时自动上传至微信图床（无需配置图床服务器）。

## 🚀 快速开始

本项目是一个**纯静态**的单页应用 (SPA)，无需复杂的安装过程。

### 方法 1：直接使用
直接下载本项目，双击打开 `index.html` 即可开始写作。

### 方法 2：部署到服务器
你可以将代码部署到任何静态托管服务（如 GitHub Pages, Vercel, Netlify）。
