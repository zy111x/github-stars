# WebTUI

Modular CSS Library that brings the beauty of Terminal UIs to the browser

Docs: at https://webtui.ironclad.sh

## Development

1. Clone the repo
2. `cd` into the repo
3. Run `bun i`
5. Run `bun run dev` to start the dev server

For feature requests / bug reports, please open an issue
