Simple and easy file uploads for React. Upload directly to any S3-compatible service with minimal setup.

Documentation is available [here](https://better-upload.com).

## Getting Started

You can add file uploads to your app in a few minutes with Better Upload. It works with any framework, like Next.js, Tanstack Start, or any other React-based framework.

To get started, follow the [quickstart guide](https://better-upload.com/docs/quickstart).

## Contributing

We welcome contributions to Better Upload!
